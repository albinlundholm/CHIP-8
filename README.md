# CHIP-8 Emulator

A CHIP-8 emulator written in C using SDL2. This project was built to explore emulator development and low-level systems programming.

## Features

* CHIP-8 instruction set implementation
* SDL2-based graphics, keyboard input, and audio
* Loads and runs standard CHIP-8 ROMs

## Requirements

* GCC (MinGW/MSYS2 UCRT64 or another C compiler)
* SDL2
* GNU Make

## Build

```sh
make
```

## Run

```sh
./chip8 roms/games/snake.ch8
```
